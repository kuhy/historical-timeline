#+TITLE: REST API
* How to execute it
  Use the maven wrapper to execute the REST API:
  #+BEGIN_SRC sh
    ../mvnw
  #+END_SRC
* How to use it
  List all users in the system using /HTTPie/:
  #+BEGIN_SRC sh
    http GET localhost:8080/pa165/rest/users
  #+END_SRC
  Or you can use =curl=:
  #+BEGIN_SRC sh
    curl -i -X GET http://localhost:8080/pa165/rest/users
  #+END_SRC
